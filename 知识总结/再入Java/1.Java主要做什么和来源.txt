任何语言都有自己擅长和不擅长的地方，Java不擅长：简化编程、服务器上的自动化管理。所以不断催发新语言的诞生，
所以现在一个项目不只使用一门语言，往往使用多门语言进行开发。

Java的特征：
1简单
2面向对象（仍然有人认为面向对象不合理，坚持面向过程）
3可移植性：同样的程序更换到不同的操作系统之后都额可以正常使用。（.net5.0完成跨平台）
4垃圾回收
5引用传递

6多线程编程支持：Java利用多线程可以有效的实现并发编程。随着版本的提高，多线程的支持也更加
到位，例如，提供有JUC环境支持（编程框架、工具类），可以简化同步异步的操作实现。

7适合分布式计算：Java的本质是为了网络开发而准备，不是应用在单机环境下，所以更多是在网络通讯的下使用，所以需要有稳定安全高效的程序支持

8安全性较高：所有的程序不是使用源代码的形式直接给出，而是字节码，所以不会被轻易破坏，但是这种文件实际可以被反编译（但是能够通过技术手段进行加密处理），
并且Java程序的运行是在虚拟机所在的资源范围中进行的，不被允许访问其他区域，所以避免的下载病毒
软件。
